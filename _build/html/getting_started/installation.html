1.2.1. System Requirements¶

  • Anaconda Distribution Python 2.7 or 3.7

  • Python must be in the system path

  • Additional Python modules

    • pyodbc
    • dropbox
    • geopandas

1.2.2. Installation¶

1.2.2.1. 1. Install Anaconda Distribution¶

https://www.anaconda.com/download/

1.2.2.2. 2. Set up database drivers (only required on macOS)¶

$ mkdir test
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew install unixodbc
$ brew install freetds --with-unixodbc

1.2.2.3. 3. Install Additional Python Modules¶

$ conda update --all
$ conda install pyodbc
$ conda install dropbox
$ conda install -c conda-forge geopandas
